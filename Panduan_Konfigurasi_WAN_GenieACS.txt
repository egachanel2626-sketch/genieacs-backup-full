üìò PANDUAN KONFIGURASI WAN, SSID, DAN INTERFACE DI GENIEACS
============================================================

üåê 1. WAN Bridge Mode
----------------------
Jika menggunakan slot WAN PPP:
- Connection Mode: PPPoE_Bridged
- Service List: INTERNET atau OTHER

Contoh untuk perangkat FiberHome:
- PPPoE Mode: PPPoE_Routed
- Service List: TR069, INTERNET
- Bind Interface:
  InternetGatewayDevice.LANDevice.1.LANEthernetInterfaceConfig.1,
  InternetGatewayDevice.LANDevice.1.WLANConfiguration.1,
  InternetGatewayDevice.LANDevice.1.WLANConfiguration.5

Jika menggunakan slot WAN IP:
- Connection Mode: IP_Bridged
- Service List: INTERNET atau OTHER


üöÄ 2. WAN Route / PPPoE
-------------------------
- Connection Mode: IP_Routed
- Service List: INTERNET


üì° 3. SSID & Keamanan Jaringan
-------------------------------
SSID dengan Password:
- Security Mode: 11i atau WPAand11i

SSID tanpa Password:
- Security Mode: None


üîó 4. Bind / Tag LAN & WLAN Interface
--------------------------------------
Isi secara manual dengan parameter di bawah ini.
Gunakan pemisah koma (,) jika lebih dari satu interface.

Khusus ZTE F670L, tambahkan WAN Interface juga pada kolom "WAN Interface".
